## Avarda Markup Shipping Suite Compatible module

* Purpose of this module is make Markup Shipping Suite module compatible with [Avarda Checkout3](https://github.com/avarda-ab/Magento-2-Checkout-3-module)

### Module Installation

1. `composer require avarda/markup-shipping-suite-compatible`  
2. `bin/magento module:enable Avarda_MarkupShippingSuiteCompatible`  
3. `bin/magento setup:upgrade`

Or 

1. Download the release zip
2. Unzip and upload content to `app/code/Avarda/MarkupShippingSuiteCompatible/`
3. `bin/magento module:enable Avarda_MarkupShippingSuiteCompatible`
4. `bin/magento setup:upgrade`
